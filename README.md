# Sabores do Brasil – Fine Dining Landing Page

A modern and responsive landing page for a Brazilian fine dining restaurant in Portugal, focused on reservations, signature dishes, and an elegant brand experience.

## Demo

You can open the `index.html` file directly in your browser or host it on any static hosting platform (GitHub Pages, Netlify, Vercel, etc.).

## Features

- **Responsive layout** optimized for mobile, tablet, and desktop.
- Sticky navigation bar with desktop and mobile menus.
- Mobile navigation with hamburger / close icon toggle and smooth scroll to sections.
- Hero section with strong call-to-action buttons for WhatsApp reservations and menu exploration.
- Sections for restaurant story, chef’s creations, menu categories, ambiance, testimonials, and contact details.
- Floating WhatsApp button to encourage direct contact and bookings.

## Tech Stack

- HTML5  
- Tailwind CSS (via CDN)  
- Vanilla JavaScript (`scripts.js` for the mobile menu behavior)  
- Google Fonts (Be Vietnam Pro & Material Symbols Outlined)
